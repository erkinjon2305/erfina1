# Moliya - Shaxsiy Moliya Boshqaruvi

Ushbu ilova shaxsiy daromad va xarajatlarni kuzatish, tahlil qilish va rejalashtirish uchun mo'ljallangan.

## Texnologiyalar
- **Frontend**: React (Vite), Tailwind CSS
- **Backend**: Firebase (Auth, Firestore)
- **Animatsiyalar**: Framer Motion
- **Grafiklar**: Recharts
- **Ikonkalar**: Lucide React

## O'rnatish

1. Firebase loyihasini yarating.
2. `.env.local` faylini yarating va Firebase kalitlarini kiriting:
   ```env
   VITE_FIREBASE_API_KEY=your_key
   VITE_FIREBASE_AUTH_DOMAIN=your_domain
   VITE_FIREBASE_PROJECT_ID=your_project_id
   VITE_FIREBASE_STORAGE_BUCKET=your_storage_bucket
   VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
   VITE_FIREBASE_APP_ID=your_app_id
   ```
3. Kutubxonalarni o'rnating:
   ```bash
   npm install
   ```
4. Ilovani ishga tushiring:
   ```bash
   npm run dev
   ```

## Xususiyatlar
- ✅ Ro'yxatdan o'tish va Kirish (Firebase Auth)
- ✅ Dashboard: Balans, Daromad, Xarajat va Jamg'arma
- ✅ Tranzaksiyalar: Qo'shish, O'chirish va Filtrlar
- ✅ Kategoriyalar: Shaxsiy kategoriyalar yaratish
- ✅ Analitika: Vizual grafiklar va tahlillar
- ✅ Hisobotlar: PDF/CSV eksport (tayyorgarlikda)
- ✅ Sozlamalar: Profil va ilova interfeysi
- ✅ To'liq O'zbek tilida
